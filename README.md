# Footage-Ingest-Agent

An experimental AI-assisted media ingest and monitoring pipeline designed to automate log analysis, detect anomalies, and generate human-readable summaries for large footage ingestion workflows.

## Goals
- Watch ingest folders automatically
- Detect completed ingest operations
- Parse structured logs
- Identify errors, warnings, or failed runs
- Discard uneventful runs automatically
- Generate concise AI summaries
- Notify users via messaging systems

## Long-Term Vision
The project explores agentic-style automation workflows where infrastructure monitoring, ingestion pipelines, and local AI systems work together to reduce repetitive manual review tasks.

## Technologies
- Python
- Docker
- Local LLMs
- NAS-based storage workflows
- JSON and log parsing
- Automation orchestration
